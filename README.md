# SSA Portfolio

A modern and responsive developer portfolio built using the latest frontend technologies including Next.js, Tailwind CSS, shadcn/ui, and Framer Motion.

Designed to showcase projects, skills, experience, and modern UI/UX development practices.

---

# Features

- Modern responsive UI
- Fast performance with Next.js
- Smooth animations
- Reusable component architecture
- SEO optimized
- Mobile-first design
- Dark / Light mode support
- Clean and scalable folder structure
- Project showcase section
- Skills & experience sections
- Contact section
- Accessible UI components

#  Installation

Clone the repository:

```bash
git clone <your-repository-url>
```

Move into the project directory:

```bash
cd ssa-portfolio
```

Install dependencies:

```bash
bun install
```

Run the development server:

```bash
bun run dev
```

Open the application:

```txt
http://localhost:3000
```
